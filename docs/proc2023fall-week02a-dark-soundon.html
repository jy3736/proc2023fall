Weekly Session Agenda and Content

Raspberry Pi

Raspberry Pi is a small but powerful device that can be used for various purposes. It is a single-board computer, which means that all the essential components are integrated on one circuit board. Raspberry Pi was created by the Raspberry Pi Foundation, a UK-based charity that aims to promote computing education and digital skills. Raspberry Pi can run different operating systems, such as Raspberry Pi OS, which is based on Debian Linux, or Ubuntu, which is another popular Linux distribution. You can use Raspberry Pi as a regular computer for browsing the web, editing documents, writing code, and playing videos.

You can also use Raspberry Pi to connect to other devices and sensors, such as LEDs, buttons, cameras, and motors. This is possible because Raspberry Pi has GPIO pins, which are special connectors that allow you to send and receive signals from external hardware. Raspberry Pi also has an HDMI port, which lets you connect it to a monitor or a TV, and USB ports, which let you plug in keyboards, mice, and other accessories. One of the most popular programming languages for Raspberry Pi is Python, which is a simple and versatile language that can be used for many applications. Python has a large and active community of developers and learners who share their projects and resources online. Python also has many libraries and modules that make it easy to work with Raspberry Pi's features and functions.

Raspberry Pi Pico w

Raspberry Pi Pico is a low-cost, high-performance microcontroller board that can run MicroPython or C/C++. It is designed to be easy to use, versatile and suitable for a wide range of applications, such as robotics, IoT, wearables and education. Unlike the Raspberry Pi 4, which is a full-fledged computer, the Raspberry Pi Pico is a simpler device that can interface with sensors, motors, LEDs and other peripherals.

The Raspberry Pi Pico has a dual-core ARM Cortex-M0+ processor, 264KB of SRAM, 2MB of flash memory, and a wide range of input/output options, including 26 GPIO pins, three analog inputs, SPI, I2C, UART and PWM. The Raspberry Pi Pico can be programmed using MicroPython, which is a Python 3 implementation for microcontrollers, or using the C/C++ SDK provided by the Raspberry Pi Foundation. The Raspberry Pi Pico is compatible with existing Raspberry Pi accessories and can also be used as a USB device or a host. The Raspberry Pi Pico is an ideal choice for anyone who wants to explore the world of microcontrollers and create amazing projects with minimal cost and complexity.

Esp32

The ESP32 is a low-cost microcontroller with integrated Wi-Fi and Bluetooth capabilities. It can be used for various purposes such as wearable devices, IoT, smart homes, and industrial automation. This microcontroller features a 32-bit Xtensa LX6 processor, which runs at up to 240 MHz with 600 DMIPS. It has a low-power co-processor that handles tasks during deep sleep mode, saving power. It has 520 KiB of RAM, 448 KiB of ROM, and supports external flash and SRAM. ESP32 supports Wi-Fi 802.11 b/g/n, Bluetooth v4.2 BR/EDR, and BLE protocols.

It can operate as a standalone system or a slave device to a host MCU through various interfaces. It also provides security features such as secure boot and flash encryption. With 34 programmable GPIOs, ESP32 can interface with ADC, DAC, SPI, I2S, I2C, UART, CAN, Ethernet, IR, and touch sensors. It has pulse counters, motor PWM, LED PWM, and an analog pre-amplifier. Programming options include Arduino IDE, Espressif IDF, MicroPython, and Lua. To conclude, the ESP32 is a multipurpose microcontroller for different applications. Its rich features and peripherals enable data processing, sensor communication, motor control, and more. Moreover, it's easy to use with a helpful developer community.

Arduino UNO

Arduino UNO is a popular microcontroller board that can be used for various projects and applications. It is based on the ATmega328P chip, which has 14 digital input/output pins, 6 analog inputs, a 16 MHz quartz crystal, a USB connection, a power jack, an ICSP header and a reset button. Arduino UNO can run on Windows, Mac OS X and Linux operating systems, and can be programmed using the Arduino IDE or other compatible software. Arduino UNO is commonly used for prototyping, robotics, interactive art, gaming, automation and more. It can communicate with sensors, actuators, displays, wireless modules and other devices using serial, SPI, I2C or other protocols. Arduino UNO is an open-source hardware and software platform that allows anyone to create and share their own inventions.

Arduino nano 33 IoT

The Arduino Nano 33 IoT is a compact yet powerful board tailored for Internet of Things (IoT) projects, seamlessly integrating wireless connectivity and energy efficiency. It employs the Arm Cortex-M0 32-bit SAMD21 processor, renowned for its computational prowess and energy-frugal operation, making it ideal for resource-efficient applications. The board is equipped with the u-blox NINA-W102 module, granting it both Wi-Fi and Bluetooth® capabilities, including Bluetooth® Low Energy (BLE) support for diverse networking options.

Notably, it incorporates an ECC608A crypto chip, ensuring secure data communication and encryption. With 14 digital I/O pins, 8 analog inputs, and 5 PWM pins, the board offers substantial flexibility. The built-in LED on pin 13 aids in quick testing and debugging. Power and programming are simplified through a micro USB port and reset button, with the board operating at 3.3V and supporting USB or external battery power sources. Compatibility with the Arduino IoT Cloud streamlines IoT project development, enabling data visualization, device control, and global project sharing. Additionally, the integrated IMU (LSM6DS3) facilitates motion tracking, positioning, navigation, and gesture recognition projects. Programming the Arduino Nano 33 IoT is effortless, thanks to support for standard Arduino libraries and compatibility with other SAMD21-compatible libraries. In summary, the Arduino Nano 33 IoT offers a cost-effective and versatile solution for a wide range of IoT applications, making it a go-to choice for IoT enthusiasts and developers.
