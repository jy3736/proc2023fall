Course Regulations / Grading Rules

上課規範 / 評分規則

  • 第一堂課鐘響 20分鐘後，不定時使用 Zuvio 線上快速簽到(點名)。
    • 線上簽到率 > 90% - 平時成績【總成績】加 5分。
  • 上課期間「干擾」0容忍!
    • 課堂上嚴禁做其他「與本課程無關」之事務，例如：寫作業、看影片、上網閒逛、...等等。
    • 罰則：第一次「口頭警告」，再犯每次『平時成績【總成績】扣 5分』。
  • 評分規則
    • 平時成績 (50%) - 課堂練習、作業、Zuvio 形成性評量 (填寫狀況)。
    • 期中成績 (20%) - 上機考試/實務專題。
    • 期末成績 (30%) - 上機考試/實務專題。

Assignment 02 - TinkerCAD - Online Arduino Simulator

  1. Please visit the website https://www.tinkercad.com/ and register for an account.
  1. This online platform can be used to develop 3D models and Arduino circuits. For this course, we will exclusively utilize the Arduino simulator.
  1. Log in to your account and create a "Circuit" named "w01-lab01." Then, use the sidebar components to complete the following design.
  1. Create two independent Arduino UNO circuits with blinking LED samples, each with a different blinking speed.
